# Easy AI Docs

5 分钟内跑通你的第一次请求。

## 开始

编辑 `docs/index.md` 即可更新首页内容。

### 第一步：获取API Key

进入 Easy AI 控制台 → 注册/登录 → 「令牌」 → 「添加新令牌」。

复制以 sk- 开头的字符串，下文统称 <您的 API Key>。


### 第 2 步：选客户端

使用方式持续更新

| 要做什么 | 推荐 |
| :--- | :---: |
| agentic编程 | ClaudeCode(推荐) |
| 使用GPT-5.5 | Codex(推荐) |
| 通用对话 GUI | OpenCode |

### 第3步：配置

已略过客户端安装，自行选择CLI或者GUI

查看对应客户端使用文档即可。
