# OpenCode

Edit `~/.config/opencode/opencode.json`. Create the file if it does not exist.

```json
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "openai": {
      "options": {
        "baseURL": "https://ai.cobitux.com/v1",
        "apiKey": "sk-dummy"
      }
    }
  },
  "model": "gpt-5.3-codex"
}
```
